Information Architecture вЂ” WordPress

Overview

WordPress.com (by Automattic) is the hosted version of the world's most popular content management system, powering over 40% of all websites. The IA must serve an extraordinarily wide audience вЂ” from personal bloggers to enterprise publishers, small business sites to full e-commerce stores. Key features include the Gutenberg block editor, thousands of themes and plugins, Jetpack integration, and a tiered hosting model. The IA balances simplicity for beginners with depth for power users.

Key pattern: WordPress has TWO admin experiences вЂ” the modern Calypso interface (React-based, at wordpress.com) and the classic wp-admin. The Gutenberg block editor is shared across both, becoming the primary content creation and site editing tool.

User Flows

Create a website

  1. Sign up в†’ choose site type (blog, business, portfolio, store)
  2. Pick a domain (free subdomain or custom)
  3. Select a theme в†’ preview
  4. Customize (colors, fonts, menus, widgets) via Site Editor
  5. Add pages and posts в†’ publish
  6. Optionally upgrade plan for more features

Write a blog post (Gutenberg)

  1. Posts в†’ Add New в†’ Gutenberg editor
  2. Add title в†’ write content using blocks (paragraph, image, heading, list, embed)
  3. Insert specialized blocks (gallery, quote, columns, table, code)
  4. Set featured image, categories, tags, excerpt
  5. Preview в†’ Publish (or schedule)

Build with blocks (Site Editor)

  1. Appearance в†’ Editor (full site editing)
  2. Edit templates (home, single post, archive, 404)
  3. Insert block patterns (pre-designed sections)
  4. Customize global styles (typography, colors)
  5. Save в†’ entire site updated consistently

Set up e-commerce

  1. Upgrade to eCommerce plan в†’ WooCommerce auto-installed
  2. Run WooCommerce setup wizard (store details, shipping, payments)
  3. Add products в†’ configure inventory, pricing, variations
  4. Customize store pages в†’ set up checkout
  5. Launch store в†’ manage orders from admin
