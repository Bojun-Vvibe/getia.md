1Password вҖ” Information Architecture

Overview

1Password is a password manager built around vaults вҖ” encrypted containers that organize credentials, documents, and secrets. The mental model is a secure digital wallet вҖ” users store logins, credit cards, notes, and identities, then autofill them in browsers/apps. Key differentiators: Watchtower (security monitoring), Travel Mode (hide vaults at borders), Secret Key (defense-in-depth beyond master password), passkey support, and developer tools (SSH keys, CLI secrets).

User Flows

Save & Autofill

Visit website вҶ’ Login form вҶ’ 1Password suggests save вҶ’ Save to vault вҶ’ Next visit вҶ’ 1Password autofill вҶ’ Done

Security Audit

Watchtower вҶ’ Review weak passwords вҶ’ Click item вҶ’ Generate new password вҶ’ Update on website вҶ’ Security score improves

Share Credential

Item Detail вҶ’ Share вҶ’ Create link (set expiry + view limit) вҶ’ Copy link вҶ’ Send to colleague вҶ’ They view once вҶ’ Link expires

Travel Mode

Settings вҶ’ Travel Mode вҶ’ Enable вҶ’ Only "safe for travel" vaults visible вҶ’ Cross border вҶ’ Disable Travel Mode вҶ’ All vaults return

SSH Key Authentication

Developer Tools вҶ’ SSH Agent вҶ’ Add Key from Vault вҶ’ Configure Git/SSH client вҶ’ Authenticate via 1Password biometrics вҶ’ SSH session established
                                                                              вҶҳ Denied вҶ’ Re-authenticate

Emergency Recovery

Family Organizer вҶ’ Members вҶ’ Select Member вҶ’ Initiate Recovery вҶ’ Member confirms via Emergency Kit вҶ’ Account recovered вҶ’ New master password set
