# Information Architecture вҖ” Deliveroo

## Overview

Deliveroo is a food delivery platform connecting customers with local restaurants, grocery stores, and its own "Editions" dark kitchens. The IA is location-first вҖ” the home screen adapts to the user's delivery address, showing nearby restaurants with estimated delivery times. The platform supports on-demand delivery (30-45 min) and scheduled delivery, with Deliveroo Plus (subscription) offering free delivery for frequent users.

## Navigation Model

- **Type**: Top nav (desktop) + bottom tab bar (mobile app)
- **Desktop Top Bar**: Deliveroo logo, Address (editable), Search, Offers, Account, Cart
- **Mobile Bottom Tabs**: Home, Search, Orders, Account
- **Cart**: Persistent floating button showing item count and total; slides up to full cart view
- **Restaurant Drill-Down**: Home вҶ’ Restaurant вҶ’ Menu вҶ’ Item customization вҶ’ Cart вҶ’ Checkout
- **Category Scroll**: Horizontal cuisine/category chips on home; vertical menu categories in restaurant

## User Flows

### Ordering Food
```
Open app вҶ’ Confirm delivery address вҶ’ Home shows nearby restaurants вҶ’ Browse by cuisine or search вҶ’ Tap restaurant вҶ’ View menu вҶ’ Select items вҶ’ Customize (size, toppings, extras) вҶ’ Add to cart вҶ’ View cart вҶ’ Enter promo code вҶ’ Add delivery instructions and tip вҶ’ Checkout вҶ’ Select payment вҶ’ Place order вҶ’ Track order: Preparing вҶ’ Rider picking up вҶ’ On the way вҶ’ Delivered вҶ’ Receive food вҶ’ Rate restaurant and rider
```

### Scheduling a Delivery
```
Browse and select items вҶ’ Go to checkout вҶ’ Tap "Delivery Time" вҶ’ Switch from "ASAP" to "Schedule" вҶ’ Select date and time slot (30-min windows, up to 7 days ahead) вҶ’ Place order вҶ’ Confirmation with scheduled time вҶ’ Reminder before delivery
```

### Grocery Shopping
```
Home вҶ’ Grocery category вҶ’ Browse stores вҶ’ Select store вҶ’ Browse categories or search products вҶ’ Add items to cart вҶ’ View cart with substitution preferences вҶ’ Checkout вҶ’ Rider shops and delivers вҶ’ Receive within 20-30 min
```

### New User Onboarding
```
Visit Deliveroo вҶ’ Sign Up (email/Google/SSO) вҶ’ Complete profile вҶ’ Guided setup wizard вҶ’ Configure preferences вҶ’ Explore key features вҶ’ Start using the product
                                                                                                                         вҶҳ Skip wizard вҶ’ Land on dashboard
```
### Manage Notifications
```
Settings вҶ’ Notifications вҶ’ Toggle email/push/in-app per category вҶ’ Set frequency (instant/daily digest/weekly) вҶ’ Save preferences
```

## Search & Filter

- **Restaurant Search**: By name, cuisine, dish name
- **Cuisine Filter**: Pizza, Sushi, Indian, Chinese, Burgers, Healthy, Vegan, etc.
- **Sort**: Recommended, Rating, Delivery Time, Distance, Delivery Fee
- **Dietary Filter**: Vegan, Vegetarian, Gluten-Free, Halal
- **Price Filter**: ВЈ / ВЈВЈ / ВЈВЈВЈ price range indicators
- **Offers Filter**: Show only restaurants with active promotions
- **Free Delivery**: Show only free delivery or Plus-eligible restaurants
- **Grocery Search**: Product search within specific stores
- **Reorder**: Quick-reorder from past orders (1-tap)

## Responsive Behavior

| Breakpoint | Behavior |
|---|---|
| Mobile App (primary) | Bottom tab bar; scrollable restaurant cards; full-screen restaurant menu; live tracking map |
| Tablet | Grid restaurant layout; expanded menu view; split cart + menu |
| Desktop Web | Horizontal restaurant grid; sticky cart sidebar on restaurant page; wider checkout |
| Mobile Web | Responsive web app; prompts to download native app; same functionality as app |

## Access Control

| Role | Capabilities |
|---|---|
| Guest (no account) | Browse restaurants and menus; must create account to order |
| Registered User | Order food, track delivery, save addresses, rate, view history, use promos |
| Plus Subscriber | Free delivery above threshold, reduced fees, member-only offers |
| Rider | Accept/reject deliveries, navigate, earn per delivery, track earnings |
| Restaurant Partner | Manage menu, accept/reject orders, view analytics, run promotions, manage hours |
| Restaurant Manager | Multi-location management, team access, financial reports |
| Deliveroo Staff | Manage platform operations, support, restaurant onboarding, pricing |
| Business Account | Company ordering, expense tracking, employee meal programs |
